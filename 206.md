# 反转链表
LeetCode第206题

## Q
反转一个单链表。
``` javascript
输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL
```

## A
``` javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var reverseList = function(head) {
    let current = head;
    let reverseHead = null;
    while(current){
        let nextTemp = current.next;
        current.next = reverseHead;
        reverseHead = current;
        current = nextTemp;
    }
    return reverseHead;
};
```
